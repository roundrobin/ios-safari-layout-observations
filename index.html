iOS Safari Layout observations

Problems observed

  • Jumping DIVs (Fixed position) when scrolling
  • On open keyboard fixed elements become relative
  • Little gap between fixed position (bottom) element and keyboard
  • Scrolling on an absolute positioned element within a scrollable parent element cause jiddery

Testing environment

  • iPhone 5 (8.3)

Tips

For a better mobile experience we recommend setting the following viewport meta tags.

<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0, minimal-ui">
